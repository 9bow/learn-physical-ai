## LeRobot이란

**HuggingFace LeRobot**은 실제 세계 로보틱스를 위한 오픈소스 라이브러리입니다. HuggingFace가 자연어처리(NLP)와 컴퓨터 비전 분야에서 모델 공유와 재사용을 대중화했듯, LeRobot은 로봇 학습(robot learning)에 같은 접근법을 적용하려 합니다.

LeRobot의 핵심 가치는 세 가지입니다:
1. **표준화된 데이터 포맷**: 다양한 로봇의 시연 데이터를 통일된 형식으로 공유
2. **사전학습 정책 허브**: HuggingFace Hub에서 바로 내려받아 사용 가능한 로봇 정책
3. **저비용 하드웨어 생태계**: 연구 접근성을 낮추는 저가 로봇 플랫폼 지원

## LeRobotDataset 포맷

### 포맷 설계 원칙

LeRobotDataset은 다음 요구사항을 충족하도록 설계되었습니다:
- 다양한 로봇(팔, 이족 보행 등)의 데이터를 동일 포맷으로 표현
- 이미지, 관절 상태, 행동 데이터를 효율적으로 저장
- HuggingFace Hub와의 원활한 통합
- PyTorch DataLoader와 호환

### 핵심 구조

```text
LeRobotDataset 디렉토리 구조:

dataset/
├── meta/
│   ├── info.json          # 데이터셋 메타데이터
│   ├── stats.json         # 관측/행동 정규화 통계
│   └── episodes.json      # 에피소드 목록
│
├── data/
│   └── chunk-000/
│       ├── episode_000000.parquet   # 관절 상태, 행동 데이터
│       ├── episode_000001.parquet
│       └── ...
│
└── videos/
    └── chunk-000/
        ├── observation.images.camera_0/
        │   ├── episode_000000.mp4
        │   └── ...
        └── observation.images.camera_1/
            └── ...
```

### Python에서 데이터셋 사용

```python
# 개념 이해용 의사코드 - LeRobotDataset 사용법

# HuggingFace Hub에서 데이터셋 로드
dataset = LeRobotDataset("lerobot/pusht_image")

# 데이터셋 정보 확인
print(f"에피소드 수: {dataset.num_episodes}")

# 단일 샘플 접근
sample = dataset[0]
# sample["observation.image"]: 이미지 텐서 [C, H, W]
# sample["observation.state"]: 관절 상태 [num_joints]

# DataLoader와 연동
from torch.utils.data import DataLoader
dataloader = DataLoader(dataset, batch_size=32, shuffle=True)
```

## 지원 정책: ACT와 Diffusion Policy

LeRobot은 최신 모방학습 알고리즘을 내장하고 있습니다.

### ACT (Action Chunking Transformer)

**ACT**는 Zhao et al.이 ALOHA 로봇 연구와 함께 개발한 모방학습 알고리즘입니다. 핵심 아이디어는 단일 행동 대신 **행동 청크(action chunk)** — 미래 여러 타임스텝의 행동 시퀀스 — 를 한 번에 예측하는 것입니다.

예를 들어, 매 스텝 하나의 관절 명령을 출력하는 대신, 한 번 추론으로 앞으로 50 스텝의 관절 명령 시퀀스를 예측합니다. 이는 Compounding Error를 줄이고 시간적 일관성을 유지하는 데 도움을 줍니다.

ACT의 아키텍처는 Transformer 기반으로, 시각 관측과 proprioceptive 상태(관절 상태)를 함께 입력받습니다.

### Diffusion Policy

**Diffusion Policy**는 8-5 챕터에서 상세히 다루며, 행동 시퀀스를 확산(diffusion) 과정으로 생성합니다. 다중 모드(multimodal) 행동 분포를 자연스럽게 처리하는 장점이 있습니다.

## 학습 워크플로

### 전체 흐름

```python
# 개념 이해용 의사코드 - ACT 학습 예시

from lerobot.common.datasets.lerobot_dataset import LeRobotDataset
from lerobot.common.policies.act.modeling_act import ACTPolicy
from lerobot.common.policies.act.configuration_act import ACTConfig

# 1. 데이터셋 로드
dataset = LeRobotDataset(
    "lerobot/aloha_sim_insertion_scripted",
    delta_timestamps={
        "observation.images.top": [0],         # 현재 이미지
        "observation.state": [0],              # 현재 관절 상태
        "action": list(range(0, 50)),          # 미래 50 스텝 행동
    }
)

# 2. 정책 초기화
config = ACTConfig(
    chunk_size=50,           # 예측할 행동 청크 길이
    n_obs_steps=1,           # 관측 히스토리 길이
    hidden_dim=512,
    dim_feedforward=3200,
    n_heads=8,
    n_encoder_layers=4,
    n_decoder_layers=7,
)
policy = ACTPolicy(config, dataset_stats=dataset.stats)

# 3. 학습 루프 (단순화)
optimizer = torch.optim.Adam(policy.parameters(), lr=1e-5)

for batch in DataLoader(dataset, batch_size=8):
    loss, info = policy.forward(batch)
    
    optimizer.zero_grad()
    loss.backward()
    optimizer.step()

# 4. 체크포인트 저장
policy.save_pretrained("outputs/act_aloha")
```

### 명령줄 학습

LeRobot은 설정 파일 기반의 명령줄 학습도 지원합니다:

```bash
# 개념 이해용 - 실제 명령줄 예시 구조
python lerobot/scripts/train.py \
  --policy.type=act \
  --dataset.repo_id=lerobot/aloha_sim_insertion_scripted \
  --training.num_epochs=200 \
  --training.batch_size=8 \
  --output_dir=outputs/my_act_policy
```

## 저가 로봇 생태계

### SO-100 / SO-101

SO-100(SO-101)은 LeRobot 생태계용 저가 단팔/양팔 로봇입니다. 저가형 서보 모터를 사용하여 비용을 크게 낮추면서도 표준 조작 과제를 수행할 수 있습니다. 하드웨어 설계가 오픈소스로 공개되어 있어 직접 조립이 가능합니다.

### Koch v1.1

Koch v1.1은 LeRobot과 함께 사용할 수 있는 또 다른 저가 단팔 로봇으로, ALOHA와 유사한 리더-팔로워(leader-follower) 구성을 지원합니다.

이런 저가 플랫폼의 등장은 로봇 모방학습 연구의 접근성을 크게 높였습니다. 수천만 원 이상의 산업용 로봇 없이도 연구 및 교육 목적으로 모방학습을 실험할 수 있습니다.

## 사전학습 모델 활용

HuggingFace Hub에 공개된 사전학습 정책을 즉시 사용하거나 파인튜닝할 수 있습니다:

```python
# 개념 이해용 의사코드 - 사전학습 정책 로드 및 평가

from lerobot.common.policies.act.modeling_act import ACTPolicy

# Hub에서 사전학습 정책 로드
policy = ACTPolicy.from_pretrained("lerobot/act_pusht_image")
policy.eval()

# 실제 로봇에서 정책 실행 (단순화)
obs = robot.get_observation()

with torch.no_grad():
    action = policy.select_action(obs)

robot.execute_action(action)
```

## 정리

LeRobot은 로보틱스 모방학습의 진입 장벽을 낮추는 오픈소스 생태계입니다. 표준화된 LeRobotDataset 포맷으로 다양한 로봇 데이터를 통합하고, ACT와 Diffusion Policy 같은 최신 알고리즘 구현을 제공하며, SO-100 같은 저가 하드웨어와 연동됩니다. HuggingFace Hub를 통한 데이터셋·모델 공유 생태계는 커뮤니티 기반 발전을 가속화하고 있습니다.
