## 프로젝트 개요

VLA(Vision-Language-Action) 모델은 Physical AI의 최전선입니다. 카메라 이미지와 자연어 지시를 함께 이해하고, 이를 로봇의 행동(관절 명령)으로 직접 변환합니다. "빨간 컵을 집어서 오른쪽 선반에 놓아"처럼 구체적인 언어 지시에 따라 로봇이 동작하는 것이 목표입니다.

이 프로젝트에서는 Isaac Lab 시뮬레이션 환경에서 시연 데이터를 수집하고, VLA 모델(NVIDIA GR00T 워크플로 또는 OpenVLA)을 파인튜닝하여 언어 지시 기반 조작을 구현하는 개념 실습을 진행합니다.

## 단계 2: 데이터 전처리 및 포맷

```python
# 시연 데이터를 LeRobot/RLDS 형식으로 변환 (개념 이해용 의사코드)
import h5py
import numpy as np
from pathlib import Path

def convert_to_lerobot_format(raw_demos_dir, output_path):
    """Isaac Lab 수집 데이터를 LeRobot 형식으로 변환"""

    demos = []
    for demo_file in Path(raw_demos_dir).glob("demo_*.hdf5"):
        with h5py.File(demo_file, 'r') as f:
            episode = {
                "observation.state": f["joint_states"][:],       # (T, 7)
                "observation.images.top": f["camera_top"][:],    # (T, H, W, 3)
                "observation.images.wrist": f["camera_wrist"][:], # (T, H, W, 3)
                "action": f["actions"][:],                         # (T, 7)
                "language_instruction": str(f.attrs["instruction"]),
            }
            demos.append(episode)

    # 통계 계산 (정규화용)
    all_actions = np.concatenate([d["action"] for d in demos])
    action_mean = all_actions.mean(axis=0)
    action_std = all_actions.std(axis=0)

    return demos, {"action_mean": action_mean, "action_std": action_std}
```

### OpenVLA 파인튜닝 대안

OpenVLA는 오픈소스 VLA로 더 자유롭게 실험할 수 있습니다.

```python
# OpenVLA 파인튜닝 (개념 예시 — 실제 코드는 공식 GitHub 확인)
from transformers import AutoModelForVision2Seq, AutoProcessor

# 사전학습 모델 로드
model = AutoModelForVision2Seq.from_pretrained(
    "openvla/openvla-7b",
    torch_dtype=torch.bfloat16,
    device_map="auto"
)
processor = AutoProcessor.from_pretrained("openvla/openvla-7b")

# LoRA 파인튜닝 설정 (전체 파라미터 대신 효율적 파인튜닝)
from peft import LoraConfig, get_peft_model

lora_config = LoraConfig(
    r=16,
    lora_alpha=32,
    target_modules=["q_proj", "v_proj"],
    lora_dropout=0.05,
    bias="none"
)
model = get_peft_model(model, lora_config)

# 학습 루프 (개념 — 실제 trainer 설정은 공식 문서 참조)
trainer = VLATrainer(
    model=model,
    args=training_args,
    train_dataset=train_dataset,
    data_collator=data_collator
)
trainer.train()
```

## 단계 4: 언어 지시 기반 추론

파인튜닝된 모델로 실제 추론을 수행합니다.

```python
# VLA 추론 노드 (개념 이해용 의사코드 — ROS 2 통합)
import rclpy
from rclpy.node import Node
from sensor_msgs.msg import Image, JointState
from std_msgs.msg import String
import torch

class VLAPolicyNode(Node):
    def __init__(self):
        super().__init__('vla_policy')

        # VLA 모델 로드 (파인튜닝된 모델)
        self.model = self._load_vla_model('/path/to/finetuned_model/')

        # 언어 지시 구독
        self.create_subscription(String, '/language_instruction',
                                 self.instruction_cb, 10)

        # 카메라 이미지 구독
        self.create_subscription(Image, '/camera/image_raw',
                                 self.image_cb, 10)

        # 관절 명령 발행
        self.action_pub = self.create_publisher(JointState,
                                                '/joint_commands', 10)

        # 추론 루프 (10Hz)
        self.create_timer(0.1, self.inference_loop)

    def inference_loop(self):
        if self.current_image is None or self.instruction is None:
            return

        # VLA 추론
        actions = self.model.predict(
            image=self.current_image,
            instruction=self.instruction
        )

        # 관절 명령 발행
        self._publish_actions(actions)
```

```bash
# VLA 정책 노드 실행
ros2 run vla_deploy policy_node

# 언어 지시 발행 (테스트)
ros2 topic pub /language_instruction std_msgs/msg/String \
  "{data: 'pick up the red cube and place it in the bin'}" --once
```

## 확장 아이디어

1. **멀티스텝 지시**: "컵을 집어서 전자레인지에 넣고 문을 닫아"처럼 여러 단계 작업
2. **대화형 수정**: 로봇 동작 중 지시를 변경하면 즉시 적응
3. **실패 복구**: 물체를 떨어뜨렸을 때 스스로 파악하고 재시도
4. **휴머노이드 이식**: Franka 팔에서 학습한 정책을 휴머노이드 상체로 이식 (cross-embodiment)
5. **시연 데이터 없는 학습**: Isaac Sim 합성 시연 + 순수 VLA 사전학습으로 소수 시연만으로 파인튜닝
