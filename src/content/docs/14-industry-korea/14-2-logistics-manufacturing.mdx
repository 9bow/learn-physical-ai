## 물류 산업이 로봇에 주목하는 이유

전자상거래의 성장으로 물류 창고는 처리해야 할 주문 수가 급증했습니다. 동시에 창고 작업자 부족과 인건비 상승이 맞물리면서, 물류 산업은 자동화에 가장 빠르게 투자하는 분야 중 하나가 되었습니다.

물류의 자동화를 어렵게 만드는 것은 **비구조화된 환경과 다양한 물체**입니다. 공장의 조립 라인은 동일한 부품이 정해진 위치에 오는 구조화된 환경이지만, 물류 창고는 수만 종의 상품이 임의의 포장 상태로 섞여 들어옵니다. 사람은 이를 쉽게 처리하지만 기존 산업 로봇은 어려웠습니다. Physical AI가 이 갭을 메우고 있습니다.

## 피킹(Picking): 핵심 과제

**피킹**은 선반에서 특정 상품을 집어 포장 상자나 컨베이어에 올려놓는 작업입니다. 물류 창고에서 가장 노동 집약적인 과제입니다.

기존 로봇 피킹 시스템은 미리 지정된 물체만 처리할 수 있었습니다. 딥러닝 기반 비전과 파악(grasping) 알고리즘의 발전으로, 처음 보는 물체도 카메라 영상에서 3D 자세(pose)를 추정하고 적절한 파지(grip) 방법을 결정하는 것이 가능해졌습니다.

Covariant, Mujin 같은 기업들은 신경망 기반 피킹 시스템을 실제 물류 창고에 배치하고 있습니다. 이러한 시스템은 수천 개 SKU(Stock Keeping Unit, 개별 상품 단위)에 대해 학습하고, 새로운 상품도 빠르게 적응합니다.

```text
┌──────────────────────────────────────────────────────────────────────┐
│                  비전 기반 피킹 시스템 파이프라인                     │
│                                                                      │
│  카메라 영상(RGB+Depth) ──▶ 물체 감지(Detection)                    │
│                                   │                                  │
│                         세그멘테이션(Segmentation)                   │
│                                   │                                  │
│                         6-DoF 자세 추정(Pose Estimation)             │
│                                   │                                  │
│                         파지 계획(Grasp Planning)                    │
│                                   │                                  │
│                         로봇 팔 실행(Execution) ──▶ 물체 이송        │
└──────────────────────────────────────────────────────────────────────┘
```

## AMR: 창고를 누비는 자율 이동 로봇

**AMR(Autonomous Mobile Robot, 자율 이동 로봇)** 은 창고 바닥을 자율적으로 이동하며 상품 선반이나 토트(tote) 상자를 이송합니다.

AMR은 이전 세대인 AGV(Automated Guided Vehicle)와 다릅니다. AGV는 바닥에 레일이나 자기 테이프를 설치하고 그 경로만 따라다녔습니다. AMR은 SLAM(Simultaneous Localization and Mapping)으로 환경 지도를 스스로 만들고, 사람이나 장애물을 실시간으로 감지하여 경로를 조정합니다.

```text
┌────────────────────────────────────────────────────────────────┐
│               AMR vs. AGV 비교                                  │
├───────────────────────┬────────────────────────────────────────┤
│ AGV                   │ AMR                                    │
├───────────────────────┼────────────────────────────────────────┤
│ 고정 경로(레일/테이프)│ 자유 이동(SLAM 기반 내비게이션)       │
│ 인프라 설치 필요      │ 인프라 변경 불필요                     │
│ 경로 변경 어려움      │ 실시간 경로 재계획                     │
│ 사람 감지 제한        │ 인식 기반 충돌 회피                    │
│ 낮은 비용             │ 높은 비용 (점차 하락 중)               │
└───────────────────────┴────────────────────────────────────────┘
```

Amazon Robotics(구 Kiva Systems)는 수십만 대의 AMR을 자사 물류 센터에 운용하고 있는 것으로 알려져 있습니다. 선반 전체를 피킹 스테이션으로 이동시키는 방식으로 작업자의 이동 거리를 크게 줄였습니다.

## 제조: 협동 로봇과 유연 자동화

기존 산업 자동화는 **고정 프로그래밍 방식**이었습니다. 자동차 조립 라인처럼 동일한 공정이 반복되는 환경에서는 효과적이지만, 제품 종류나 공정이 바뀌면 재프로그래밍에 큰 비용이 듭니다.

**협동 로봇(collaborative robot, cobot)** 은 안전 펜스 없이 사람 옆에서 함께 작업할 수 있는 로봇으로, 중소기업도 도입 가능한 가격과 프로그래밍 편의성이 특징입니다. Universal Robots, Fanuc, KUKA 등이 주요 공급사입니다.

Physical AI의 등장으로 협동 로봇의 유연성이 더 높아지고 있습니다. 비전 기반 피드백으로 부품 위치가 달라져도 적응하거나, 모방 학습으로 작업자가 손으로 시연한 동작을 그대로 복제할 수 있습니다.

## 비전 기반 품질 검사

제조 과정에서 결함 제품을 골라내는 **품질 검사(quality inspection)** 는 전통적으로 숙련된 검사원이 육안으로 수행했습니다. 딥러닝 기반 비전 시스템이 이를 대체하고 있습니다.

합성곱 신경망(CNN) 기반 이상 탐지(anomaly detection) 모델은 정상 제품 이미지를 학습하고, 스크래치, 찌그러짐, 오염 같은 이상을 자동으로 탐지합니다. 피로 없이 24시간 운용 가능하고, 일관성 있는 판단을 내린다는 장점이 있습니다.

한계도 있습니다. 결함 유형이 다양하거나 새로운 결함이 나타나면 재학습이 필요하고, 세밀한 검사(마이크로미터 수준의 치수 측정)에는 특수 센서와 추가 엔지니어링이 필요합니다.

## 물리 AI가 바꾸는 공장의 미래

지금까지의 자동화는 **구조화된 환경에서 정해진 과제만** 수행했습니다. Physical AI가 가져오는 변화는 **비구조화된 환경에서 유연하게 작업**하는 능력입니다.

이 변화가 완전히 실현되면 공장의 설계 철학 자체가 달라집니다. 제품 라인 변경 시 생산 라인 전체를 재구성하는 대신, 로봇에게 새로운 과제를 가르치는 것으로 전환할 수 있습니다. "재프로그래밍 비용"이 극적으로 낮아지는 것입니다.

이 전환이 완전히 이루어지기까지는 안전성 검증, 비용 절감, 데이터 누적의 시간이 필요합니다. 현재는 구조화된 단순 반복 과제에서 시작하여 점진적으로 복잡도를 높이는 방식으로 실적을 쌓아가는 단계입니다.
