## GR00T 프로젝트란

**NVIDIA Isaac GR00T(Generalist Robot 00 Technology)** 는 NVIDIA가 추진하는 휴머노이드 로봇(humanoid robot)용 파운데이션 모델 프로젝트입니다. 이름의 "GR00T"는 범용 로봇 기술을 뜻하며, 숫자 "00"은 두 눈을 형상화하기도 합니다.

휴머노이드 로봇은 기존 산업용 로봇 팔과 근본적으로 다른 어려움을 안고 있습니다. 두 다리로 균형을 잡으면서, 두 팔로 다양한 물체를 조작하고, 동시에 주변 환경과 상호작용해야 합니다. 자유도(degrees of freedom)가 수십 개에 달하고, 전신의 협조 동작이 필요합니다.

GR00T는 이 복잡성을 해결하기 위해 두 가지 핵심 전략을 사용합니다.

1. **대규모 합성 데이터 생성**: Isaac Sim을 이용해 다양한 환경·과제에서 합성 시연 데이터를 대량 생성합니다.
2. **사람 동작 모방학습**: 인간의 동작 캡처(motion capture) 데이터를 로봇 정책 학습에 활용합니다.

## GR00T N1: 첫 번째 공개 모델

**GR00T N1**은 NVIDIA가 공개한 GR00T 프로젝트의 첫 번째 모델입니다. 이 모델은 두 가지 핵심 구성 요소로 이루어집니다.

```text
┌──────────────────────────────────────────────────────────────────┐
│                   GR00T N1 아키텍처 개요                          │
│                                                                  │
│  ┌─────────────────────────────────┐                            │
│  │        인지 모듈                │  ← 다중 카메라 입력         │
│  │  (Perception Module)            │     언어 명령               │
│  │  VLM 기반 장면 이해             │     로봇 고유감각           │
│  └────────────────┬────────────────┘                            │
│                   │                                              │
│                   ▼                                              │
│  ┌─────────────────────────────────┐                            │
│  │        행동 모듈                │  → 관절 각도 목표           │
│  │  (Action Module)                │     전신 동작 계획          │
│  │  흐름 매칭 기반 정책            │     실시간 제어             │
│  └─────────────────────────────────┘                            │
└──────────────────────────────────────────────────────────────────┘
```

**인지 모듈**은 카메라 이미지, 언어 지시, 고유감각(proprioception) 데이터를 통합하여 현재 상황을 이해합니다. **행동 모듈**은 그 이해를 바탕으로 실행 가능한 관절 궤적(joint trajectory)을 생성합니다.

## 합성 데이터 생성 워크플로

GR00T 학습의 핵심 병목은 **데이터 부족**입니다. 실제 휴머노이드 로봇으로 다양한 과제를 수행하는 시연 데이터를 수집하려면 막대한 시간과 비용이 필요합니다. NVIDIA는 이 문제를 **합성 데이터(synthetic data)** 로 해결합니다.

**Isaac Sim 기반 합성 데이터 파이프라인:**

```text
┌─────────────────────────────────────────────────────────────────┐
│              합성 데이터 생성 파이프라인                         │
│                                                                 │
│  1. 환경 무작위화(Randomization)                                │
│     ├─ 물체 위치·방향 무작위                                    │
│     ├─ 조명·텍스처·배경 변경                                    │
│     └─ 카메라 위치 다양화                                       │
│                           │                                     │
│                           ▼                                     │
│  2. 궤적 생성                                                   │
│     ├─ 모션 캡처 → 로봇 리타게팅(retargeting)                   │
│     ├─ 강화학습으로 기본 동작 습득                              │
│     └─ 전문가 시연 데이터 추가                                  │
│                           │                                     │
│                           ▼                                     │
│  3. GR00T 학습                                                  │
│     ├─ 합성 데이터 + 실제 데이터 혼합                          │
│     ├─ 모방학습(Imitation Learning)                             │
│     └─ 강화학습으로 추가 개선                                   │
└─────────────────────────────────────────────────────────────────┘
```

## 모션 캡처와 리타게팅

사람의 몸동작은 로봇에게 가장 자연스러운 시연 데이터 원천입니다. 사람이 물건을 집는 동작을 모션 캡처 수트로 기록하면, 그 궤적을 로봇의 관절 구조로 변환하는 **리타게팅(retargeting)** 과정이 필요합니다.

사람과 로봇의 링크(link) 길이, 관절 가동 범위(joint range), 자유도 배치가 다르기 때문에 단순 복사가 아닌 최적화 문제를 풀어야 합니다.

NVIDIA의 **Isaac Lab**은 이 리타게팅 파이프라인을 지원하는 도구를 포함하며, 모션 캡처 마커 데이터를 로봇 관절 목표값으로 변환하는 역운동학(inverse kinematics, IK) 솔버를 제공합니다.

## Isaac Lab·Isaac Sim과의 연계

GR00T는 NVIDIA의 전체 Physical AI 플랫폼 위에서 동작합니다.

**Isaac Sim** 역할:
- 실사 수준의 렌더링으로 시각 데이터 합성
- 물리 시뮬레이션으로 접촉·충돌 시뮬레이션
- 도메인 무작위화(domain randomization)로 Sim-to-Real 갭 축소

**Isaac Lab** 역할:
- GR00T 학습을 위한 강화학습 환경 제공
- GPU 병렬 환경으로 대규모 롤아웃 수집
- 평가 프레임워크 및 체크포인트 관리

## 휴머노이드 파트너십 생태계

NVIDIA는 GR00T를 자사 기술 스택으로만 닫아두지 않고, 여러 휴머노이드 로봇 제조사와 파트너십을 맺어 실제 하드웨어에서 검증을 진행합니다. 1X Technologies, Agility Robotics, Apptronik, Boston Dynamics, Figure AI, Fourier Intelligence, Sanctuary AI, Unitree 등 다양한 기업들이 GR00T 플랫폼과 협력한다고 발표했습니다.

이 파트너십 모델은 두 가지 이점을 제공합니다.

- **NVIDIA 입장**: 다양한 로봇 형태에서 GR00T를 검증하고 데이터를 확보할 수 있습니다.
- **로봇 제조사 입장**: NVIDIA의 시뮬레이션·학습 인프라를 활용하여 개발 속도를 높일 수 있습니다.

## 현재 상태와 전망

GR00T 프로젝트는 지속적으로 발전 중입니다. GR00T N1은 초기 공개 모델이며, 이후 버전에서는 더 복잡한 양손 조작, 모바일 매니퓰레이션(이동+조작), 사람과의 직접 상호작용 시나리오로 확장될 것으로 예상됩니다.

주의할 점은 GR00T가 일반적인 연구 코드베이스가 아니라 NVIDIA 플랫폼에 긴밀히 통합된 상업 제품이라는 것입니다. 공개된 기술 문서와 논문을 통해 원리를 학습할 수 있지만, 전체 학습 파이프라인은 내부적으로 운영됩니다.

다음 챕터에서는 GR00T와 함께 자주 언급되는 NVIDIA Cosmos 월드 파운데이션 모델을 살펴봅니다.
