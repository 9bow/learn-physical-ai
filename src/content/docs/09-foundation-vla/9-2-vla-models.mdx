## VLA 모델이란

**VLA(Vision-Language-Action) 모델**은 시각 입력(카메라 이미지), 언어 명령(자연어 지시), 그리고 로봇의 상태(현재 관절 각도 등)를 함께 입력받아, 다음 수행할 **행동(action)** 을 출력하는 신경망 정책입니다.

이름 그대로 세 가지 모달리티를 통합합니다.
- **V(Vision)**: RGB 카메라 이미지, 깊이 이미지 등
- **L(Language)**: "컵을 오른쪽 컨테이너에 넣어" 같은 자연어 지시
- **A(Action)**: 엔드이펙터(end-effector) 위치 변화량, 그리퍼 개폐, 관절 각도 등

VLA의 핵심 아이디어는 **행동을 토큰으로 표현**하여, 언어 모델이 다음 단어를 예측하듯 다음 행동을 예측하게 만드는 것입니다.

```text
┌──────────────────────────────────────────────────────────────────┐
│                  VLA 모델의 입출력 구조                           │
│                                                                  │
│  입력                        처리                   출력         │
│  ──────                      ────                   ────         │
│  [이미지 패치] ──┐                                               │
│                  ├──▶ 통합 트랜스포머 ──▶ [행동 토큰 시퀀스]    │
│  [언어 토큰] ────┤    (Vision + LLM)      → 연속값 디코딩       │
│                  │                         → Δx, Δy, Δz         │
│  [로봇 상태] ────┘                         → 그리퍼 개폐         │
└──────────────────────────────────────────────────────────────────┘
```

## RT-2: 로봇 트랜스포머(Robotics Transformer) (Google DeepMind)

**RT-2(Robotic Transformer 2)** 는 Google DeepMind가 발표한 VLA 모델로, 대형 시각-언어 모델(VLM)을 로봇 행동 예측에 직접 활용한 선구적 연구입니다.

**핵심 설계:**
- 기존 웹 데이터로 사전학습된 PaLI-X 또는 PaLM-E 같은 대형 VLM을 베이스로 사용합니다.
- 로봇 행동(엔드이펙터 6DoF 위치, 그리퍼 값)을 **문자열 토큰**으로 인코딩합니다. 예를 들어 x 좌표 0.034를 "034" 같은 토큰으로 표현합니다.
- 로봇 시연 데이터를 웹 데이터와 혼합하여 파인튜닝합니다.

**의의:** 인터넷에서 학습한 상식 지식이 로봇 과제 수행에 그대로 전이됩니다. RT-2는 학습 중 본 적 없는 물체("멸종한 동물을 집어")나 추상적 명령도 이해하는 능력을 보였습니다.

**한계:** 모델 크기가 수십~수백억 파라미터 수준으로 추론 속도가 느리며, 학습 데이터와 모델 가중치가 공개되지 않았습니다.

## OpenVLA: 오픈소스 VLA

**OpenVLA**는 스탠퍼드 주도 공동 연구팀(UC Berkeley·Google DeepMind·MIT·TRI 참여)이 공개한 오픈소스 VLA 모델입니다. RT-2의 아이디어를 재현하면서 학습 코드, 모델 가중치, 데이터셋을 모두 공개하여 연구자들이 자유롭게 활용·개조할 수 있게 했습니다.

**핵심 설계:**
- **Prismatic VLM**(시각 인코더 + LLM 결합)을 베이스 아키텍처로 사용합니다.
- Open X-Embodiment 데이터셋으로 사전학습합니다.
- 행동을 256개 이산 빈(bin)으로 양자화(quantization)하여 기존 LLM 어휘에 추가합니다.
- 소비자용 GPU에서도 파인튜닝 가능한 크기(약 70억 파라미터 수준)를 목표로 합니다.

**LoRA 파인튜닝 지원:** 파라미터 효율 파인튜닝(Parameter-Efficient Fine-Tuning, PEFT) 기법인 LoRA(Low-Rank Adaptation)를 사용하면 단일 GPU로도 특정 환경에 맞게 파인튜닝할 수 있습니다.

```python
# 개념 이해용 의사코드 — 실제 OpenVLA API와 다를 수 있습니다
from openvla import OpenVLAModel

model = OpenVLAModel.from_pretrained("openvla/openvla-7b")

# 추론: 이미지 + 언어 지시 → 행동
observation = {
    "image": camera_frame,          # (H, W, 3) RGB 이미지
    "instruction": "pick up the red cup",
}
action = model.predict_action(observation)
# action = {"delta_xyz": [0.01, -0.02, 0.03], "gripper": 0.8}
```

## Octo: 경량 범용 정책

**Octo**는 버클리 대학교(UC Berkeley) 연구팀이 공개한 소형 범용 로봇 정책입니다. VLM 기반이 아닌 **트랜스포머 + 확산 모델(Diffusion Policy)** 아이디어를 결합하는 등 다양한 실험을 포함합니다.

**핵심 설계:**
- 상대적으로 작은 파라미터 수로 빠른 추론을 지향합니다.
- 이미지, 언어, 로봇 상태 등 다양한 입력 모달리티를 **토큰 시퀀스로 균일하게 처리**합니다.
- Open X-Embodiment 데이터로 사전학습되었으며, 새 로봇 팔에 소량 데이터로 파인튜닝이 가능합니다.
- **확산 헤드(diffusion head)** 와 **회귀 헤드(regression head)** 중 선택 가능합니다.

Octo의 의의는 새로운 로봇 구조(embodiment)에 파인튜닝하는 **전이 학습(transfer learning)** 파이프라인을 명확히 제시했다는 점입니다.

## π0 (파이제로): Physical Intelligence

**π0(파이-제로)** 는 스타트업 Physical Intelligence가 공개한 VLA 모델입니다. π는 원주율 기호에서 유래한 이름으로, Python과는 무관합니다. 이 회사는 로봇 범용 정책 개발을 전문으로 하며, π0은 다양한 **이중 팔 조작(bimanual manipulation)** 과제에 초점을 맞춥니다.

**핵심 설계:**
- **흐름 매칭(flow matching)** 기반 확산 프로세스를 행동 생성에 사용하여 고차원·연속적 행동 분포를 더 잘 모델링합니다.
- VLM 베이스(PaliGemma 등)와 행동 전문 디코더를 결합하는 **하이브리드 아키텍처**를 사용합니다.
- 세탁물 접기, 조립, 테이블 정리 등 복잡한 장기 과제(long-horizon task)를 목표로 합니다.
- 언어 명령에 조건화된 행동과 조건화되지 않은 행동을 모두 지원합니다.

## 행동 표현 방식 비교

행동을 어떻게 표현하느냐는 VLA 모델 설계의 핵심 선택입니다.

**이산 토큰화(discrete tokenization):**
- 연속값을 정해진 빈으로 양자화합니다.
- 기존 LLM 어휘 체계를 그대로 활용할 수 있어 구현이 간단합니다.
- 양자화 오차가 정밀한 조작에서 문제가 될 수 있습니다.

**확산 기반(diffusion-based):**
- 행동을 연속 확률 분포로 모델링합니다.
- 다봉 분포(multimodal distribution)를 표현할 수 있어, 여러 방법으로 수행 가능한 과제에 유리합니다.
- 추론 시 디노이징(denoising) 스텝이 필요하여 계산 비용이 높습니다.

**회귀(regression):**
- 직접 연속값을 출력합니다.
- 가장 단순하지만, 행동 분포가 복잡할 때 평균값으로 수렴하는 문제가 있습니다.
