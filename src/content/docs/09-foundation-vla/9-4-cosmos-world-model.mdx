## Cosmos란 무엇인가

**NVIDIA Cosmos**는 NVIDIA가 개발한 **월드 파운데이션 모델(World Foundation Model, WFM)** 플랫폼입니다. 단순한 비디오 생성 모델이 아니라, 물리 세계의 동작 원리를 이해하고 미래 상태를 예측·생성할 수 있는 대형 생성 모델을 가리킵니다.

월드 모델(world model)이라는 개념은 AI가 세계의 내부 표현을 가지고 시뮬레이션할 수 있어야 한다는 아이디어에서 출발합니다. 인간은 공을 던지기 전에 그 공이 어디 떨어질지 머릿속으로 시뮬레이션합니다. Cosmos는 이와 유사하게, **현재 상태와 행동이 주어졌을 때 그 결과를 시각적으로 생성**합니다.

**기존 이미지/비디오 생성 모델과 Cosmos의 차이:**

```text
┌───────────────────────────────────────────────────────────────┐
│              일반 비디오 생성 vs Cosmos WFM                    │
├───────────────────────┬───────────────────────────────────────┤
│   일반 비디오 생성    │   Cosmos WFM                          │
├───────────────────────┼───────────────────────────────────────┤
│ 미적 품질 최적화      │ 물리적 일관성 최적화                  │
│ 텍스트 → 비디오       │ 상태 + 행동 → 다음 상태(비디오)       │
│ 물리 법칙 무관        │ 물체 이동·충돌·중력 존중               │
│ 창의적 표현           │ 예측 정확성과 일반화                   │
│ 엔터테인먼트 목적     │ Physical AI 학습 가속화 목적           │
└───────────────────────┴───────────────────────────────────────┘
```

## 왜 월드 모델이 필요한가

Physical AI 시스템, 특히 로봇을 학습시키려면 막대한 양의 상호작용 데이터가 필요합니다. 실제 로봇으로 데이터를 수집하면 다음 문제가 발생합니다.

- **시간 비용**: 로봇이 실시간으로 동작하므로 대규모 수집이 느립니다.
- **하드웨어 비용**: 여러 대의 물리 로봇을 24시간 운용하는 비용이 큽니다.
- **안전 문제**: 학습 초기 단계에서 로봇이 환경을 손상시킬 수 있습니다.
- **다양성 한계**: 실제 환경에서 희귀한 시나리오를 충분히 수집하기 어렵습니다.

Cosmos는 이 문제를 **생성적 합성 데이터(generative synthetic data)** 로 해결하려 합니다. 학습된 월드 모델이 다양한 가상 시나리오의 시각적·물리적으로 일관된 데이터를 생성하면, 이를 로봇 정책 학습에 활용할 수 있습니다.

## Cosmos의 구성 요소

Cosmos 플랫폼은 여러 핵심 구성 요소로 이루어집니다.

### 1. 비디오 토크나이저 (Video Tokenizer)

비디오 프레임을 압축된 잠재 표현(latent representation)으로 변환합니다. 고해상도 비디오를 직접 다루면 계산 비용이 크기 때문에, 토크나이저가 먼저 정보를 요약합니다.

```text
┌─────────────────────────────────────────────────────────────┐
│               Cosmos 비디오 처리 파이프라인                  │
│                                                             │
│  원본 비디오 ──▶ 토크나이저 ──▶ 잠재 토큰 ──▶ WFM         │
│  (고해상도)      (압축)         (저차원)      (생성/예측)   │
│                                    │                        │
│                                    ▼                        │
│                  디토크나이저 ◀── 출력 토큰                 │
│                  (재구성)                                   │
│                    │                                        │
│                    ▼                                        │
│               합성 비디오 (새 환경·시나리오)                │
└─────────────────────────────────────────────────────────────┘
```

Cosmos는 이산 토크나이저(discrete tokenizer)와 연속 토크나이저(continuous tokenizer) 두 종류를 제공하며, 용도에 따라 선택합니다.

### 2. 월드 파운데이션 모델 (World Foundation Model)

토큰화된 비디오 시퀀스를 입력받아 다음 상태를 예측하는 대형 트랜스포머 모델입니다. 사전학습 시 수백만 시간 분량의 실세계 및 시뮬레이션 비디오를 사용하여 물리적 인과관계를 학습합니다.

### 3. 조건부 생성 (Conditional Generation)

- **텍스트 조건부**: "로봇 팔이 컵을 집어 트레이에 놓는다"는 설명으로 해당 동작 비디오를 생성합니다.
- **행동 조건부**: 로봇의 관절 명령 시퀀스를 조건으로 결과 비디오를 생성합니다.
- **첫 프레임 조건부**: 초기 장면 이미지에서 이후 전개를 생성합니다(비디오 예측).

## Physical AI 학습 가속화

Cosmos가 Physical AI 에코시스템에서 담당하는 역할을 구체적으로 살펴봅니다.

**시나리오 1: 합성 학습 데이터 생성**

Isaac Sim에서 로봇이 기본 과제를 수행하는 비디오를 소량 생성한 뒤, Cosmos로 이를 다양하게 변형합니다. 다른 조명, 다른 물체 배치, 다른 로봇 스킨 등을 자동 생성하여 학습 데이터 다양성을 높입니다.

**시나리오 2: 정책 평가**

학습된 로봇 정책이 생성한 행동 시퀀스를 Cosmos에 입력하면, 그 결과가 어떤 모습일지 비디오로 시각화합니다. 물리 시뮬레이터 없이도 정책의 결과를 빠르게 평가할 수 있습니다.

**시나리오 3: 데이터 증강**

실제 로봇 시연 비디오를 Cosmos로 편집하여 배경·물체 외관을 변경하는 데이터 증강(data augmentation)을 수행합니다. 실제 수집 비용을 유지하면서 다양성을 높입니다.

## Cosmos의 물리적 일관성

Cosmos가 일반 비디오 생성 모델과 다른 핵심은 **물리적 일관성(physical consistency)** 입니다. 이를 위해 사전학습 데이터에 다음이 포함됩니다.

- 실세계 조작 영상(로봇 팔, 공장 자동화 등)
- 자율주행 차량 영상
- 시뮬레이터(Isaac Sim 등)에서 생성한 물리 기반 영상

완벽한 물리 법칙 준수는 어렵지만, 생성된 비디오가 물체 낙하, 충돌, 미끄러짐 등의 기본 물리 현상을 그럴듯하게 표현할 수 있습니다.

또한 NVIDIA API 카탈로그를 통해 클라우드 추론 서비스 형태로도 접근 가능합니다.
