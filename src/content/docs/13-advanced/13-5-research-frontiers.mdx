## 물리 AI 연구의 현재 위치

이전 챕터들에서 다룬 기술들—강화학습, 모방 학습, VLA 모델, 도메인 랜덤화, 미분가능 시뮬레이션, neural rendering—은 모두 물리 AI의 핵심 도구입니다. 그러나 어떤 전문가도 "물리 AI 문제가 해결됐다"고 말하지 않습니다. 이 챕터에서는 현재 연구 공동체가 씨름하고 있는 최전선 문제들과 향후 5년의 전망(이하 전망 내용은 현재 연구 동향에 기반한 추측임)을 살펴봅니다.

## 월드 모델(World Models)

**월드 모델**은 로봇이 내부적으로 "세계가 어떻게 작동하는가"를 시뮬레이션하는 능력입니다. 과거의 관측에서 미래 상태를 예측하고, 자신의 행동이 환경에 어떤 영향을 미칠지 상상할 수 있다면, 실제로 행동하기 전에 계획을 검증할 수 있습니다.

인간은 컵을 밀기 전에 컵이 넘어질지, 얼마나 미끄러울지 직관적으로 예측합니다. 이 능력은 수년간의 물리 세계 경험에서 나옵니다. 로봇에게 이와 같은 내재적 물리 이해를 부여하는 것이 월드 모델 연구의 목표입니다.

```text
┌────────────────────────────────────────────────────────────────────┐
│                    월드 모델의 역할                                 │
│                                                                    │
│  현재 관측(o_t) ──▶ 월드 모델 ──▶ 미래 예측(o_{t+1}, o_{t+2}...) │
│                         ▲                                          │
│                    행동(a_t) 입력                                  │
│                                                                    │
│  활용 예시:                                                         │
│  • 정책이 행동 후보를 상상으로 평가                                │
│  • 희소한 보상에서 예측 오차로 탐색                                │
│  • 시뮬레이터 없이 내부 롤아웃(rollout)                            │
└────────────────────────────────────────────────────────────────────┘
```

현재 물리 AI에서 월드 모델 연구의 주요 도전은 **접촉이 포함된 조작의 정확한 예측**입니다. 사람이 물체를 집거나 미는 장면처럼 접촉 역학이 복잡한 상황을 몇 초 앞서 예측하는 것은 아직 미해결 문제입니다. Google DeepMind의 Genie 2 같은 대규모 영상 모델이 물리적으로 그럴듯한 환경 시뮬레이션 능력을 보이고 있지만, 아직 정밀한 로봇 제어에 직접 쓸 수 있는 수준은 아닙니다.

## 대규모 로봇 데이터의 문제

언어 모델이 인터넷의 텍스트 수조 토큰으로 학습한 것처럼, 로봇 파운데이션 모델도 대규모 다양한 로봇 경험 데이터를 필요로 합니다. 그러나 로봇 데이터 수집에는 언어 데이터와 근본적으로 다른 어려움이 있습니다.

**물리적 수집 비용**: 언어 데이터는 이미 인터넷에 존재합니다. 로봇 데이터는 실제 로봇을 움직이며 기록해야 합니다. 사람이 로봇 팔을 조작하여 시연 데이터를 수집하는 것은 시간과 비용이 많이 듭니다.

**다양성의 한계**: 특정 연구실에서 수집한 데이터는 그 환경, 조명, 물체 종류에 편향됩니다. 진정한 일반화를 위해서는 수백~수천 곳의 다양한 환경 데이터가 필요합니다.

**데이터 표준화**: 로봇마다 관절 수, 센서 구성, 좌표 체계가 다릅니다. 서로 다른 로봇의 데이터를 통합하는 표준(예: Open X-Embodiment 데이터셋 형식)이 필요합니다.

현재 연구 방향으로는 **합성 데이터(synthetic data)** — 시뮬레이터에서 대규모로 생성 — 와 실제 데이터의 결합, **인터넷 영상에서 로봇 정책 학습**(사람이 물체를 다루는 영상에서 조작 기술 추출), **텔레오퍼레이션(teleoperation) 효율화** 등이 연구되고 있습니다.

## Generalist 휴머노이드 정책

**Generalist 휴머노이드 정책**은 단일 모델이 다양한 가정 내 과제(청소, 요리 보조, 물건 정리)를 자연어 명령에 따라 수행하는 것을 목표로 합니다. 이는 현재 연구의 가장 야심찬 목표 중 하나입니다.

현재 상태를 솔직하게 평가하면, 공개된 시연 영상들은 인상적이지만 대부분 통제된 환경에서 특정 과제를 수행하는 것입니다. 예상치 못한 물체 배치, 낯선 조명, 새로운 과제가 주어지면 성능이 크게 저하되는 것이 일반적입니다. 그러나 학습 데이터와 모델 규모가 커질수록 일반화 능력이 향상되는 경향이 관찰되고 있어, 스케일링 법칙(scaling law)이 로보틱스에도 적용될 가능성에 대한 연구가 활발합니다.

## Sim-to-Real 격차의 근본적 해소

sim-to-real 갭은 물리 AI의 가장 오래된 문제 중 하나입니다. 현재 주류 접근인 도메인 랜덤화와 적응 방법들은 갭을 줄이지만 완전히 없애지는 못합니다.

근본적 해소를 위한 연구 방향들이 있습니다.

**초고품질 시뮬레이터**: 물리 엔진의 정확도 자체를 높여 갭의 원인을 줄이는 방향. 액추에이터 모델, 접촉 역학, 센서 노이즈 시뮬레이션의 정밀도 향상.

**온라인 적응(online adaptation)**: 실물 배포 후에도 새로운 관측으로 지속적으로 정책을 업데이트하는 방법. 안전 제약 하에서 실물 탐색을 허용하는 방식.

**Real-to-sim-to-real**: 실제 환경을 neural rendering으로 정밀하게 시뮬레이터에 재현한 뒤, 그 시뮬레이터에서 학습하여 다시 실물로 전이하는 방식. 시각적 갭은 크게 줄일 수 있지만 물리 갭은 여전히 남습니다.

## Embodied Reasoning

**Embodied reasoning(체화 추론)** 은 언어나 텍스트만으로는 획득할 수 없고, 물리 세계와의 실제 상호작용을 통해서만 형성되는 추론 능력을 지칭합니다.

"무거운 물체를 들 때 몸의 균형을 어떻게 잡는가", "좁은 공간을 통과하기 위해 어떻게 몸을 비트는가", "물체가 미끄러지기 시작할 때 그립을 어떻게 조정하는가" 같은 지식은 텍스트로 설명하기 어렵고, 실제로 경험해야 체득됩니다.

대형 언어 모델(LLM)은 많은 물리 상식을 텍스트로 학습했지만, 실제 감각운동(sensorimotor) 경험이 없습니다. 물리 세계와 상호작용하면서 이러한 경험을 축적한 모델이 진정한 embodied reasoning을 갖출 수 있다는 가설 하에 연구가 진행 중입니다. 이것이 데이터 규모와 다양성 문제와 직결됩니다.

## 향후 5년 전망 (추측 포함)

다음은 현재 연구 동향을 기반으로 한 전망이며, 불확실성이 높습니다.

```text
┌───────────────────────────────────────────────────────────────────┐
│               물리 AI 향후 5년 전망 (추측)                         │
├──────────────────────┬────────────────────────────────────────────┤
│ 단기 (1~2년)         │ 중기 (3~5년)                               │
├──────────────────────┼────────────────────────────────────────────┤
│ 특정 물류 과제에서   │ 다과제 가정 내 로봇 초기 상용화 시도      │
│ 휴머노이드 실증 배포 │ 실시간 월드 모델 기반 계획                 │
│ 증가                 │ 대규모 합성+실제 데이터 결합 표준화        │
│ VLA 모델 경량화      │ 안전 보장 형식화 일부 영역 진전            │
│ (엣지 추론 가능)     │ 물리 이해 능력의 질적 개선                 │
└──────────────────────┴────────────────────────────────────────────┘
```

**낙관적 시나리오**: 스케일링 법칙이 로보틱스에도 강하게 작용하여 대규모 데이터와 모델로 generalisation 문제가 빠르게 해결된다.

**보수적 시나리오**: 물리 세계의 복잡성은 언어 도메인과 근본적으로 다르며, 데이터 수집 병목과 안전 요건으로 인해 진전이 예상보다 느리다.

현실은 두 시나리오 사이 어딘가일 가능성이 높습니다. 특정 구조화된 환경(공장, 물류 창고)에서는 빠른 진전이 일어나는 반면, 비구조화 환경(가정, 야외)에서의 완전한 자율 로봇은 더 오랜 시간이 필요할 것으로 보입니다.

## 이 분야에서 연구자로 기여하려면

물리 AI 연구는 여러 분야의 교차점에 있습니다. 기계공학, 제어이론, 컴퓨터 비전, 머신러닝, 컴퓨터 그래픽스 중 어느 한 분야라도 깊이 있는 이해가 있다면 기여할 지점이 있습니다.

현재 특히 수요가 높은 기여 영역은 다음과 같습니다.
- 실물 로봇 데이터 수집 효율화(텔레오퍼레이션, 합성 데이터)
- 더 정확한 접촉 시뮬레이션 및 미분가능 역학 모델
- 안전 보장을 갖춘 학습 기반 제어
- 시각적 사실성과 물리 정확성을 동시에 갖춘 real-to-sim 파이프라인
- 소형/저가 하드웨어에서 실행 가능한 경량화 VLA 모델

물리 AI는 인공지능이 디지털 세계를 넘어 물리 세계로 확장되는 전환점에 있습니다. 이 분야의 발전은 제조, 물류, 의료, 서비스 로봇 전반에 광범위한 영향을 미칠 것입니다.
